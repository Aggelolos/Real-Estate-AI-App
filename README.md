My App is a Real Estate Portfolio Manager.

This Web Application will allow a Realtor/Agent to safely Log In, Save a Detail for Each Listing an Agent has, and Use Artificial Intelligence to Predict a Fair Selling Price.

The User of this application is a Realtor/Agent. The Problem that my application will solve is assisting the Realtor-Agent in Organizing Listings and Automatically Estimating Property Values Without Guesswork.

To accomplish the artificial intelligence piece of my application, I will utilize Prediction (Specifically Linear Regression) to provide an Automated Estimate of House Prices Based Upon Historical Data. As opposed to utilizing a Human to Provide a Simple Rule-Based Estimate, the Artificial Intelligence Will Learn from Past Data to Provide a Much More Accurate Estimate.

The Input Variable(s) to the Model will include variables such as Area Income, House Age, Number of Bedrooms, etc. And the Output Variable Will be the Predicted House Price.

I Will Utilize Mock Data From the House_Price.csv File That I Previously Worked With.
